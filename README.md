# 💳 Crédito Inteligente

Crédito Inteligente es un proyecto web diseñado para presentar una plataforma de servicios financieros de manera moderna, intuitiva y accesible. Su objetivo es ofrecer una experiencia visual atractiva donde los usuarios puedan conocer los diferentes productos y servicios financieros disponibles.

## 🚀 Características

- Diseño moderno y responsivo.
- Interfaz intuitiva y fácil de navegar.
- Presentación de servicios financieros.
- Secciones organizadas para una mejor experiencia de usuario.
- Arquitectura modular utilizando componentes reutilizables.

- Crear una interfaz moderna y profesional.
- Mejorar la experiencia del usuario.
- Facilitar la navegación entre los diferentes servicios.
- Aplicar buenas prácticas de desarrollo Frontend.
- Mantener una estructura organizada y escalable.

## ⚙️ Instalación

Clona el repositorio:

```bash
git clone https://github.com/usuario/credito-inteligente.git
```

Ingresa al proyecto:

```bash
cd credito-inteligente
```

Instala las dependencias:

```bash
npm install
```

Ejecuta el servidor de desarrollo:

```bash
npm run dev
```

Abre en el navegador:

```
http://localhost:3000
```
